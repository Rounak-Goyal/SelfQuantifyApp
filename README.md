Self Quantified App to help keep users tracks of their daily activity.
Using Flask and SQlite  

Reqirements-
1)flask  
2)flask_sqlalchemy  
3)matplotlib   
  
To run setup: 
  pip install flask  
  pip install sql_alchemy  
  pip install matplotlib  
  python app.py
